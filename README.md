# python-RPG

This is a personal project with the intent of making a text-based RPG using Python and other various tools as necessary.

Title: The Lord's Favour

Description:
This game tells the story of a young man who has been hired by the local lord to clear out a dangerous and intricate dungeon. You will choose your character name, class, and abilities. As well as collect powerful loot along the way.
And who knows, perhaps you will discover a danger unknown to the land of mortals?

Languages Used:
Python
